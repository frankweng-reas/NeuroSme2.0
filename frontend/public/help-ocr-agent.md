# OCR / Vision 使用說明

## Overview

**OCR / Vision** 是 AI 驅動的文件欄位抽取工具，專為批量處理固定格式文件而設計。上傳圖片或 PDF 後，AI 依據預先定義的欄位設定，自動辨識並擷取指定資料，結果可直接編輯後複製或匯出為 CSV。

| 典型使用情境 | 說明 |
|-------------|------|
| 名片批量建檔 | 上傳名片圖片，自動抽取姓名、公司、電話、Email |
| 發票 / 收據處理 | 擷取發票號碼、日期、金額、供應商等欄位 |
| 報名表 / 申請表 | 從掃描表單中提取填寫內容 |
| 物流標籤 | 讀取追蹤號碼、地址、收件人等資訊 |
| 產品標籤 | 抽取 SKU、規格、條碼等資料 |

> **與 Chat 的差異**：Chat 適合自由問答；OCR / Vision 適合「大量、重複、固定格式」的文件資料擷取，並可搭配批量模式一次處理多份檔案。

---

## 快速流程：選擇設定 → 上傳圖片或 PDF → 確認並儲存結果

---

## 1. 左側：設定管理

### 建立抽取設定
- 點擊「**＋ 新增設定**」，輸入設定名稱（如：名片、發票），選擇文件類型標籤，並定義要抽取的欄位。
- 可套用**內建範本**快速建立常用設定（範本提供名片、發票等預設欄位）。
- 建立後點擊列表中的設定，即切換至該設定的操作介面。

### 欄位設定
每個欄位需填寫**欄位名稱**（如：公司名稱、發票金額），AI 會依欄位名稱從文件中識別對應資料。

### 設定操作（⋯ 選單）
| 操作 | 說明 |
|------|------|
| **編輯** | 修改設定名稱、欄位清單 |
| **刪除** | 刪除該設定（歷史記錄不受影響） |

---

## 2. 右側：上傳與抽取

### 單筆上傳
1. 點擊「**上傳圖片**」或直接拖放，支援 `.jpg`、`.jpeg`、`.png`、`.webp`、`.pdf`。
2. AI 自動分析並填入各欄位值，可在結果區直接修改。
3. 點擊「**儲存**」將結果存入歷史記錄。

### 批量上傳
1. 點擊「**批量處理**」按鈕，進入批量模式。
2. 選取多份圖片或 PDF，系統依序逐一辨識。
3. 全部完成後可直接在表格內編輯，並一次**複製**或**匯出 CSV**。

---

## 3. 結果操作

| 操作 | 說明 |
|------|------|
| **直接編輯** | 點擊欄位值即可修改 |
| **儲存** | 將目前結果存入歷史記錄 |
| **複製** | 將所有欄位值複製至剪貼簿（可選擇是否含欄位名稱） |
| **歷史記錄** | 查看過去所有抽取結果，可依時間軸或表格檢視，並可刪除特定記錄 |

---

## 4. 圖片預覽

- 右側上方顯示原始圖片預覽，點擊「**放大**」可進入全螢幕 Lightbox。
- Lightbox 支援滾輪縮放與拖曳平移，方便比對原始文件與抽取結果。
- 左右拖曳中間分隔線可調整圖片與欄位的顯示比例。

---

## 注意事項

- 圖片解析度越高，辨識準確率越好；低解析度或模糊圖片可能造成欄位遺漏或錯誤。
- PDF 會取第一頁進行辨識；多頁 PDF 建議先拆頁後再批量上傳。
- 批量處理中若某筆失敗，其餘不受影響，失敗項目會以紅色標示。
- 歷史記錄儲存於伺服器，重新登入後仍可查看。
