# Doc Refiner 使用說明

## 功能定位

Doc Refiner 以 AI 自動將 PDF 文件整理為結構化的 **Q&A 問答集** 或 **條列摘要**，整理結果可直接編輯、調整後下載為排版精美的 PDF，也可做為知識庫的素材。

---

## 使用流程

### 第一步：上傳文件

| 項目 | 說明 |
|------|------|
| **支援格式** | PDF（最大 20 MB） |
| **上傳方式** | 拖曳至上傳區，或點擊選擇檔案 |
| **整理模式** | Q&A（問答集）或 摘要（條列重點） |
| **使用模型** | 留空時使用租戶預設模型；建議選 GPT-4o 或 Gemini 以取得最佳品質 |

點擊「**開始智慧整理**」，AI 將自動閱讀 PDF 並產出結果（通常需 10–30 秒）。

---

### 第二步：比對 & 編輯

- **左側** 顯示原始 PDF 預覽，方便對照確認。
- **右側** 顯示 AI 整理結果，每一條均可直接點擊編輯。

| 操作 | 說明 |
|------|------|
| **編輯文字** | 直接點擊問題/答案/摘要欄位修改 |
| **刪除條目** | 滑鼠移至卡片右上角，點「🗑」刪除 |
| **新增條目** | 點底部「＋ 新增一條」 |
| **修改標題** | 右側頂部的「文件標題」欄可修改（用於 PDF 標題頁） |
| **重新上傳** | 點左上角「重新上傳」，回到第一步 |

---

### 第三步：下載 PDF

確認內容無誤後，點擊右上角「**下載 PDF**」。

系統會產生包含封面標題與所有整理條目的 PDF，支援繁體中文輸出，可直接對外分享或列印。

---

## 整理模式說明

| 模式 | 適合情境 | 輸出格式 |
|------|----------|----------|
| **Q&A 模式** | 客服問答、教育訓練、知識庫素材 | 問題 + 答案 |
| **摘要模式** | 技術文件、報告、規章條款 | 章節標題 + 重點摘要 |

---

## 注意事項

- 文件字數過多時，AI 可能無法一次處理全部內容，建議分批上傳。
- 掃描式 PDF（純圖片）無法提取文字，請使用含文字層的 PDF。
- 整理結果僅供參考，建議人工審閱後再使用。
