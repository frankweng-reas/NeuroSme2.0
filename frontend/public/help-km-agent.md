# 知識管理助理使用說明

## 快速流程：上傳文件 → 勾選 → 提問

---

## 1. 文件管理（左欄）

### 文件狀態

| 狀態 | 說明 |
|------|------|
| 處理中 | 上傳後系統正在進行切塊與向量化，請稍候 |
| 就緒 | 向量化完成，可參與 RAG 檢索 |
| 錯誤 | 處理失敗，可刪除後重新上傳 |

### 文件分組

左欄文件依類型分為兩個群組：

- 🔒 **私人文件**（金色左邊框）：僅自己可看到與存取，上傳後預設勾選。
- 🌐 **公用文件**（藍色左邊框）：所有成員共用，需管理員（admin / manager）才能上傳或刪除。

### 上傳文件

- 支援格式：**PDF、純文字（.txt）、Markdown（.md）**。
- 點擊「上傳文件」按鈕選擇檔案，可一次上傳多個。
- 上傳時可選擇文件類型，有助於提昇向量化品質。

### 勾選文件

- 只有**已勾選且狀態為「就緒」**的文件才會納入本次對話的 RAG 檢索範圍。
- 每個群組（私人 / 公用）右上角有獨立的**全選 / 全不選**按鈕。
- 頂端也有**全域全選 / 全不選**可一次操作所有文件。
- **勾選狀態會自動記憶**，重新整理頁面後仍會保留上次的勾選設定。

---

## 2. 對話區（右側）

- 在輸入框輸入問題，按 Enter 或送出按鈕發送。
- AI 會從**勾選文件**中進行語意檢索，找出相關段落後回答。
- **清除對話**：點對話區右上角「清除」圖示，清空目前的對話紀錄。

---

## 3. AI 設定區（右側面板）

- 可調整 **AI 模型、回應語言、回應風格（詳細 / 精簡）、角色** 等參數。
- 設定僅對目前工作階段有效，不影響其他使用者。
- 詳細說明請參閱「AI 設定」線上說明。

---

## 注意事項

- 未勾選任何文件時，送出問題會提示先勾選文件。
- 刪除公用文件需管理員以上權限；私人文件僅本人可刪除。
- 大型文件（> 10 MB）向量化時間較長，請等候狀態變為「就緒」再提問。
