# Writing Agent - 商業文書撰寫使用說明

## 快速流程：填寫左側表單 → 點擊「生成草稿」→ 在右側編輯或複製

---

## 1. 個人資訊（左側上方）

- 填入您的**姓名**與**公司名稱**，AI 生成時會自動帶入署名與落款。
- 填過一次後系統會記憶，下次開啟無需重填，可點擊展開修改。

---

## 2. 文件設定

### 文件類型
選擇要撰寫的文件種類：

| 類型 | 說明 |
|------|------|
| 商業 Email | 跟進報價、會議邀請、感謝拜訪等商務郵件 |
| 提案簡介 | 產品或服務的提案說明文件 |
| 工作報告 | 月報、專案進度等業績或工作彙整 |
| 會議摘要 | 記錄討論重點、決議事項與行動項目 |
| 內部公告 | 辦公室公告、人事異動等全員通知 |
| 自訂文件 | 自行指定文件類型，如感謝函、道歉信、合作邀約等 |

### 套用範本（選填）
- 部分文件類型提供預設範本，套用後會自動填入欄位，可直接修改後生成。

---

## 3. 填寫資訊

- 依選擇的文件類型填入對應欄位（收件對象、目的、語氣、關鍵內容等）。
- 欄位越完整，AI 生成的草稿越貼近需求。
- **對 AI 的額外要求**（選填）：可補充特殊指示，例如字數限制、特定用語或格式要求。

---

## 4. 生成草稿

- 點擊底部金色「**生成草稿**」按鈕，AI 會根據填入的資訊即時產生文件。
- 右側頂端可切換 **AI 模型**，模型偏好會自動記憶。
- 生成中可等候完成；若需重新生成，直接再次點擊「生成草稿」即可覆蓋。

---

## 5. 編輯與格式工具

草稿生成後，可在右側編輯器直接修改內容，並使用工具列進行格式調整：

| 按鈕 | 功能 |
|------|------|
| **B** | 粗體 |
| *I* | 斜體 |
| H2 | 標題 |
| 條列 / 數字清單 | 清單格式 |
| ↩ | 復原上一步 |

---

## 6. AI 局部改寫

- 在編輯器中**選取文字**後，工具列會出現 AI 改寫選項：

| 按鈕 | 說明 |
|------|------|
| 重寫 | 保留語意，換一種表達方式 |
| 縮短 | 保留核心意思，縮減篇幅 |
| 正式化 | 調整為更正式的語氣 |
| 友善化 | 調整為更親切友善的語氣 |
| 自訂 | 輸入自訂改寫指令 |

---

## 7. 複製與清除

- **複製**：將草稿全文以純文字複製到剪貼簿，可直接貼入 Email、文書軟體或通訊工具。
- **清除**：清空右側編輯器，準備重新生成。

---

## 注意事項

- 左側設定欄（個人資訊、語氣偏好）會儲存於本機瀏覽器，重新整理後仍保留。
- 草稿不會自動儲存，離開頁面前請複製或另存。
- 若尚未在後台設定 AI 模型，「生成草稿」按鈕將無法使用。
