# 商業洞察助理

## **1. 建立主題 -> 2.匯入資料 -> 3.開始對話**

### 分析主題（左側）

- **選擇或建立主題**：每個主題對應一組要分析的資料範圍（與資料模型／匯入設定綁定）。
- **主題旁的資料庫圖示**：
  - **綠色**：已匯入資料（可進行分析對話）。
  - **紅色**：尚無資料（需先匯入）。
  - **琥珀色**：正在確認資料狀態。
- **開啟匯入**：點該圖示，或使用主題列選單中的「匯入資料」。
---
### 匯入資料

- 在匯入／資料管理流程中選擇**資料模型（範本）**，並依提示上傳 **CSV**。
- 完成後，左側與對話列上的資料狀態會更新；有資料後即可針對該主題提問。
---
### 對話區

- **範例問題**：管理你的範例問題，方便日後重複使用。
- **資料庫**：顯示目前主題的資料庫筆數、點擊後開啟匯入資料相關操作。
- **清除對話**：清空目前對話紀錄。

---
### 範例問題

- **系統提供**：可瀏覽後**複製**文字再自行貼上。
- **我的範例**：可**一鍵帶入**輸入框後再送出。
