- 筆記可能格式不規整，請主動發想讀者可能會問的問題，將零散內容整合成有意義的問答對

- 允許將多段相關筆記合併為一個問答對，只要邏輯連貫
- 不要只逐字複述，要歸納整理成有用的答案
